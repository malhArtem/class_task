## Лабораторная работа: «Изучение конфигурации Windows 10 (скриншот-отчёт)»

**Цель работы:**  
Ознакомиться с основными инструментами конфигурирования Windows 10, выполнить безопасные для личного компьютера действия по просмотру и изменению простых параметров, зафиксировать результаты в виде скриншотов для отчёта.

**Оборудование и ПО:**  
- Личный компьютер с Windows 10 (любая редакция).  
- Учётная запись с правами администратора (для некоторых действий).  
- Средство создания скриншотов (например, «Ножницы», Snip & Sketch, PrintScreen).

**Важно:** Все изменения, предлагаемые в работе, легко отменяются или не влияют на стабильность системы. Вы можете выполнять их на своём основном ПК без риска.

---

## Общее задание (список скриншотов)

Сделайте **скриншоты** перечисленных ниже окон и вставьте их в отчёт с кратким пояснением.

1. **Сведения о системе** – окно `msinfo32`.  
2. **Локальные пользователи и группы** – два скриншота:  
   - список существующих пользователей (оснастка «Управление компьютером»);  
   - окно создания локального пользователя (без учётной записи Microsoft) с заполненными полями (самого пользователя можно не создавать, достаточно показать заполненную форму).  
3. **Локальная групповая политика** – окно `gpedit.msc` (если Windows 10 Pro/Enterprise) или запись «Home – недоступно».  
4. **Конфиденциальность** – настройки микрофона и камеры.  
5. **Центр обновления Windows** – главная страница и дополнительные параметры.  
6. **Брандмауэр** – список правил для входящих подключений.  
7. **Контроль учётных записей (UAC)** – окно настройки уровня уведомлений.  
8. **Точка восстановления** – создание точки (с подтверждением успеха).  
9. **Персонализация** – временная смена обоев рабочего стола (скриншот рабочего стола).

## Подробное описание каждого шага (для выполнения)

### 1. Сведения о системе
- Нажмите `Win + R`, введите `msinfo32`, нажмите «ОК».  
- Сделайте скриншот окна «Сведения о системе» (должны быть видны версия ОС, сборка, тип системы).

### 2. Локальные пользователи и группы (включая создание локального пользователя)
#### 2.1. Просмотр существующих пользователей
- Нажмите правой кнопкой мыши по «Пуск» → «Управление компьютером».  
- В левом меню: «Служебные программы» → «Локальные пользователи и группы» → «Пользователи».  
- Сделайте скриншот списка пользователей (ничего не меняйте).

#### 2.2. Создание локального пользователя (без учётной записи Microsoft) – достаточно скриншота окна
- Откройте **Параметры** (`Win + I`) → **Учётные записи** → **Семья и другие пользователи** (или «Другие пользователи»).  
- Нажмите **«Добавить пользователя»**.  
- В появившемся окне «Учётная запись Microsoft» внизу нажмите ссылку **«У меня нет данных для входа этого пользователя»**.  
- В следующем окне нажмите внизу **«Добавить пользователя без учётной записи Microsoft»**.  
- Заполните поля (можно вымышленными данными):  
  - Имя пользователя (например, `Lab_Local`)  
  - Пароль (например, `P@ssw0rd`)  
  - Подтверждение пароля  
  - Подсказка для пароля (например, `пароль для лабы`)  
- **Сделайте скриншот этого заполненного окна** (кнопку «Далее» или «Создать» можно **не нажимать** – для отчёта достаточно показать форму).  
- *Примечание:* если вы всё же создадите пользователя, после работы можете его удалить в том же разделе «Семья и другие пользователи».

### 3. Локальная групповая политика
- Нажмите `Win + R`, введите `gpedit.msc`, «ОК».  
- Если редактор открылся – сделайте скриншот любого раздела (например, «Конфигурация компьютера → Административные шаблоны → Система»).  
- Если у вас Windows 10 Home – напишите в отчёте: «Редакция Home, gpedit.msc недоступен» и скриншот сообщения об ошибке.

### 4. Параметры конфиденциальности (микрофон и камера)
- **Параметры** (`Win + I`) → **Конфиденциальность и безопасность** → **Микрофон** – скриншот.  
- Затем **Камера** – скриншот.  
- Ничего не меняйте, если не хотите.

### 5. Центр обновления Windows
- **Параметры** → **Центр обновления Windows** – скриншот главной страницы.  
- Нажмите «Дополнительные параметры» – скриншот (например, раздел «Выберите, когда следует устанавливать обновления»).

### 6. Брандмауэр Windows (список правил)
- Нажмите `Win + R`, введите `wf.msc`, «ОК».  
- Слева выберите **«Правила для входящих подключений»**.  
- Сделайте скриншот части списка (10–15 строк).

### 7. Контроль учётных записей (UAC)
- Нажмите `Win + R`, введите `UserAccountControlSettings`, «ОК».  
- Сделайте скриншот ползунка уровня уведомлений.  
- По желанию временно измените уровень (например, на «Всегда уведомлять»), сделайте ещё один скриншот, затем верните исходный.

### 8. Создание точки восстановления
- В поиске «Пуск» введите «Создание точки восстановления» → откройте **Свойства системы** (вкладка «Защита системы»).  
- Выберите системный диск (C:) → «Настроить» – убедитесь, что защита **включена** (если нет – включите, сделайте скриншот).  
- Нажмите «Создать» → введите имя (например, `Моя_точка_дата`) → «Создать».  
- После успеха сделайте скриншот сообщения «Точка восстановления создана успешно».

### 9. Персонализация (временная смена обоев)
- Правой кнопкой по рабочему столу → **Персонализация** → **Фон**.  
- Временно измените обои на любую другую стандартную картинку или цвет.  
- Сделайте скриншот рабочего стола с новыми обоями.  
- После выполнения работы верните исходные обои (можно позже).

## Контрольные вопросы (ответить письменно)

1. Для чего нужна оснастка `msinfo32`? Какую информацию о системе можно из неё получить?  
2. Чем отличается локальный пользователь от учётной записи Microsoft?  
3. Почему в Windows 10 Home отсутствует редактор локальной групповой политики `gpedit.msc`?  
4. Какие настройки конфиденциальности (микрофон, камера, местоположение) можно отключить без вреда для большинства программ?  
5. Что такое точка восстановления системы и как её использовать для отката изменений?  
6. Какие действия требуют подтверждения UAC? Что произойдёт, если установить уровень уведомлений «Никогда не уведомлять»?  
7. Как проверить, включён ли брандмауэр Windows, без открытия оснастки `wf.msc`?  
8. В чём преимущество создания локального пользователя без учётной записи Microsoft?
