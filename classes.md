# Классы и объекты в Python

## Что такое класс?

**Класс** - это шаблон или чертеж для создания объектов. Он определяет:
- Какие данные будут храниться (атрибуты)
- Какие действия можно выполнять (методы)

Представьте, что класс - это форма для печенья, а объекты - это само печенье, созданное по этой форме.

## Что такое объект (экземпляр)?

**Объект** - это конкретная реализация класса. Это отдельная сущность, созданная по шаблону класса.

Если класс `Dog` - это общее понятие "собака", то объект `my_dog = Dog("Бобик", 3)` - это конкретная собака с именем Бобик и возрастом 3 года.

## Подробный пример

```python
# Класс - чертеж автомобиля
class Car:
    def __init__(self, brand, color, year):
        # Атрибуты - характеристики объекта
        self.brand = brand      # марка
        self.color = color      # цвет
        self.year = year        # год выпуска
        self.is_running = False # состояние
    
    # Методы - действия, которые может выполнять объект
    def start_engine(self):
        self.is_running = True
        return f"Двигатель {self.brand} запущен"
    
    def stop_engine(self):
        self.is_running = False
        return f"Двигатель {self.brand} остановлен"
    
    def get_info(self):
        return f"{self.brand}, {self.color}, {self.year} год"

# Создание объектов - конкретных автомобилей
car1 = Car("Toyota", "красный", 2020)  # объект 1
car2 = Car("BMW", "синий", 2022)       # объект 2

print(car1.get_info())  # Toyota, красный, 2020 год
print(car2.get_info())  # BMW, синий, 2022 год

print(car1.start_engine())  # Двигатель Toyota запущен
```

## Ключевые понятия

### 1. Конструктор `__init__()`
- Вызывается автоматически при создании объекта
- Инициализирует атрибуты объекта
- `self` - ссылка на текущий объект

```python
def __init__(self, brand, color):
    self.brand = brand  # создаем атрибут brand
    self.color = color  # создаем атрибут color
```

### 2. Атрибуты
- Переменные, которые хранят данные объекта
- У каждого объекта свои значения атрибутов

```python
car1 = Car("Toyota", "красный")
car2 = Car("BMW", "синий")

print(car1.brand)  # Toyota
print(car2.brand)  # BMW - другой объект, другое значение
```

### 3. Методы
- Функции, которые определяют поведение объекта
- Первый параметр всегда `self`

```python
def start_engine(self):
    # self позволяет обращаться к атрибутам этого объекта
    self.is_running = True
```

## Еще один пример: Студент

```python
class Student:
    def __init__(self, name, age, major):
        self.name = name
        self.age = age
        self.major = major
        self.grades = []
    
    def add_grade(self, grade):
        self.grades.append(grade)
    
    def get_average(self):
        if not self.grades:
            return 0
        return sum(self.grades) / len(self.grades)
    
    def get_info(self):
        return f"{self.name}, {self.age} лет, {self.major}"

# Создаем студентов
student1 = Student("Анна", 20, "Информатика")
student2 = Student("Иван", 19, "Математика")

# Каждый объект независим
student1.add_grade(5)
student1.add_grade(4)
student2.add_grade(3)

print(student1.get_info())      # Анна, 20 лет, Информатика
print(student1.get_average())   # 4.5
print(student2.get_average())   # 3.0
```

## Почему использовать классы?

1. **Инкапсуляция** - объединение данных и методов в одной сущности
2. **Повторное использование** - один класс можно использовать для создания многих объектов
3. **Организация кода** - логическая группировка функциональности
4. **Моделирование реального мира** - объекты программы соответствуют реальным сущностям

## Разница между классом и объектом

| Класс | Объект |
|-------|--------|
| Шаблон, чертеж | Конкретная реализация |
| `class Car:` | `my_car = Car()` |
| Определяет структуру | Содержит реальные данные |
| Один на программу | Много в программе |

```python
# Класс - общее понятие
class Phone:
    def __init__(self, model, os):
        self.model = model
        self.os = os

# Объекты - конкретные телефоны
phone1 = Phone("iPhone 14", "iOS")  # объект 1
phone2 = Phone("Samsung S23", "Android")  # объект 2
```

# Создание классов в Python

Классы в Python позволяют создавать собственные типы данных, объединяя данные и функциональность.

## Базовый синтаксис

```python
class ClassName:
    # Конструктор (инициализатор)
    def __init__(self, параметры):
        # Атрибуты экземпляра
        self.attribute = значение
    
    # Методы
    def method_name(self, параметры):
        # код метода
```



## Пример простого класса

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def bark(self):
        return f"{self.name} говорит: Гав!"
    
    def get_info(self):
        return f"{self.name}, {self.age} лет"
```

## Использование класса

```python
# Создание экземпляров
dog1 = Dog("Бобик", 3)
dog2 = Dog("Шарик", 5)

# Вызов методов
print(dog1.bark())  # Бобик говорит: Гав!
print(dog2.get_info())  # Шарик, 5 лет
```
