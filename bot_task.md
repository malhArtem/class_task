## **Техническое задание: Telegram-бот "Денежное дерево"**

### **1. Общее описание**
Разработать Telegram-бота, где пользователь может "выращивать" виртуальное дерево. Каждый день пользователь может выполнить команду, чтобы его дерево выросло на случайное количество сантиметров

### **2. Основные функции**

#### **2.1. Ежедневный рост дерева**
- **Команда `/grow`** — основной функционал
- **Работа команды:**
  1. Проверка, использовал ли пользователь уже попытку сегодня
  2. Если нет — генерация случайного роста (от 1 до 30 см)
  3. Увеличение общей высоты дерева
  4. Показ результата пользователю
- **Ограничение:** 1 использование в сутки на пользователя
- **Сообщение при повторе:** Уведомление о необходимости ждать до следующего дня


#### **2.2. Просмотр дерева**
- **Команда `/tree`** — просмотр своего дерева
- **Отображение:**
  - Текущая высота в см/метрах

#### **2.3. Рейтинг**
- **Команда `/top`** — топ-10 пользователей
- **Формат вывода:**
  - Список топ-10 по высоте дерева
  - Отображение имен (username или first_name)
  - Форматированная высота (см/м)
  - Позиция текущего пользователя в общем рейтинге
  - Общее количество участников

#### **2.4. Вспомогательные команды**
- **`/start`** — регистрация, приветствие, инструкция

### **4. Команды бота (полный список)**
```
/start  - Начало работы, регистрация
/grow   - Вырастить дерево (1 раз в день)
/tree   - Посмотреть своё дерево
/top    - Топ-10 самых высоких деревьев
```
