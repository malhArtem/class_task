### **Игра: "Виселица" (Hangman)**
**Суть:** Классическая игра на угадывание слова по буквам 
**Пример**:
`_е_е_ра_` - *телеграм*

**Хендлеры:**
1. `/start` — Начать новую игру.
2. `/letter а` — Угадать букву.
3. `/word телеграм` — Попытка угадать сразу слово.
4. `/surrender` — Сдаться и показать слово.

**Дополнительно:**
- Тематические категории слов
- ASCII-рисунок виселицы при ошибках
- Счетчик побед/поражений


---

**Рекомендации по выполнению:**
1. Сначала спроектируйте таблицу в БД.
2. Напишите функции для работы с БД (добавление, выборка, обновление).
3. Затем "привяжите" эти функции к хендлерам.
4. Не забывайте фильтровать данные по `user_id`, чтобы пользователи видели только свои данные.
